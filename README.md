# iyzico-mcp

> MCP server scaffold for Iyzico (Turkish payment processor).

**v0.0.1 — package scaffold.** This repository contains the bootstrap project structure (TypeScript + Bun, `@modelcontextprotocol/sdk`, Biome, CI). Tools are not yet implemented.

## Stack

- TypeScript 5.5+ on Bun (Node 20+ compatible)
- [`@modelcontextprotocol/sdk`](https://www.npmjs.com/package/@modelcontextprotocol/sdk) for the server runtime
- [`iyzipay`](https://www.npmjs.com/package/iyzipay) — official Iyzico Node SDK (planned wrapper target)
- Biome for lint + format
- Bun's built-in test runner
